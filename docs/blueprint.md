# **App Name**: FinSight

## Core Features:

- Landing Page: Clean, simple, and modern landing page with 'Get Started' button.
- Dashboard Layout: Dashboard with sidebar navigation: Analysis, Live News, and FinBot tabs.
- Analysis Section: Analysis tab with Overview, Trend Analysis, and Backtest Simulator sections, plus a stock search and analyze button.
- Stock Overview Cards: Cards displaying key stock data: Current Value, Market Cap, PE Ratio, etc.
- Stock Data Graph: Graph to display historical stock data (API integration point).
- FinBot Integration: Integrate Gemini API to provide financial query assistance; the Gemini tool will assist people with their financial queries, incorporating outside information as needed.
- API Integrability: Ensure the entire project is designed for easy integration with external APIs for data retrieval and processing.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) for a sense of trust and sophistication.
- Background color: Off-white (#F9FAFB) for a clean, light theme.
- Accent color: Light blue (#60A5FA) to highlight interactive elements and calls to action.
- Body and headline font: 'Inter' sans-serif for a modern, clean, and readable UI.
- Use minimalist icons to represent different financial metrics and analysis tools.
- Maintain a clean and structured layout, with clear separation between sections using whitespace and subtle dividers.
- Subtle transitions and animations when loading data or navigating between sections to enhance user experience.
